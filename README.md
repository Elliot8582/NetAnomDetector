# NetAnomDetector
Ein Python-basiertes Tool zur Erkennung von unüblichen Aktivitäten in Netzwerken. Das Tool nutzt die Bibliotheken Scapy für das Sniffen von Paketen und Isolation Forest zur Erkennung von Anomalien im Netzwerkverkehr. Es hilft, verdächtige Netzwerkaktivitäten zu überwachen und zu identifizieren.
